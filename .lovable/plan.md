## التفاصيل التقنية

### إعداد Google OAuth في Supabase
بعد الموافقة على الخطة، ستحتاج إلى:

1. **إعداد مشروع Google Cloud:**
   - الذهاب إلى Google Cloud Console
   - إنشاء OAuth Client ID
   - إضافة Authorized redirect URLs

2. **إعداد Supabase:**
   - تفعيل Google Provider في Authentication → Providers
   - إدخال Client ID و Client Secret
   - التأكد من إعداد Site URL و Redirect URLs
